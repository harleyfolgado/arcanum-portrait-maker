# Security

## Supported Versions

Only the latest public release is supported.

## Reporting Issues

Use GitHub Issues for reproducible installer, restore, or portrait workflow problems. Do not post game install contents, account credentials, or machine-specific file paths in public issues.

## File Handling

Arcanum Portrait Maker is designed to write only to the selected Arcanum install folder after showing a preview and creating a restore point. Starter Character Pack restore uses restore data created by the app.
