# Arcanum Portrait Maker v1.0.0

This release introduces the validated Starter Character Pack installer.

## Highlights

- Replaces all 12 stock pregenerated Pick Character slots with 12 validated starter characters.
- Keeps Create Character custom portrait installation separate.
- Creates a restore point before Starter Character Pack installation.
- Restores stock characters from app-created restore data.
- Detects old Create Character starter portrait entries without removing them automatically.
- Includes the full 12-character public portrait set in this repository.
- Directs app downloads to the GitHub Release asset instead of the repository ZIP.
- Confirms no Create Character leakage from the Starter Character Pack.
- Confirms validated starting gear for the release roster.

## Validated Release Summary

- Runtime-validated on May 7, 2026.
- Clean GOG/base install tested.
- Install and uninstall support are part of the release flow.
- Optional UAP extra content is not part of `v1.0.0` support.
