# Arcanum Portrait Maker v1.0.0

This release introduces the Starter Character Pack installer.

## Highlights

- Replaces all 12 stock pregenerated Pick Character slots.
- Keeps Create Character custom portrait installation separate.
- Creates a restore point before Starter Character Pack installation.
- Restores stock characters from app-created restore data.
- Detects old Create Character starter portrait entries without removing them automatically.
- Includes the full 12-character public portrait set in this repository.
- Directs app downloads to GitHub Releases instead of the repository ZIP.

## Before Installing

Use a copied Arcanum install for first-time testing. Download the Windows ZIP from GitHub Releases, extract it, and run `ArcanumPortraitMaker.exe`. The app does not include Arcanum itself or any official game executable.

## Manual Test Checklist

- Preview Starter Character Pack before install.
- Install Starter Character Pack.
- Confirm preview reports `Installed 12 / 12`.
- Launch Arcanum manually.
- Click through all 12 Pick Character slots.
- Start Sir Rogers, Elias, Mara, and one small-race character.
- Verify starter inventories.
- Test Restore Stock Characters.
