# Arcanum Portrait Maker v1.1.0

Release title:
`Arcanum Portrait Maker v1.1.0 - Restore Female Smallfolk`

## Highlights

- Adds `Restore Female Smallfolk` to restore two missing female smallfolk Pick Character slots through loose-file install and rollback.
- Stella is the restored Female Gnome replacement for slot `16090`.
- Loretta is the restored Female Halfling replacement for slot `16091`.
- The app previews the packaged Stella and Loretta portraits before install.
- The app backs up and restores whatever was previously installed in slot 4 and slot 5.

## Scope

- Restore Female Smallfolk replaces two Pick Character slots only.
- Clean GOG/base Arcanum is supported.
- The 12-character Starter Character Pack remains supported and compatible.
- No UAP requirement was added for this feature.
- No DAT patching is used.
- No `data/Players` payload is used.
- No scripts are installed.
- No Create Character support was added for Stella or Loretta.

## Install Notes

1. Open `Restore Female Smallfolk`.
2. Select the Arcanum install folder.
3. Review the replacement preview.
4. Install Stella and Loretta.
5. Open Arcanum manually and use `Single Player -> New Game -> Pick Character`.

## Restore Notes

- Restore uses the backup created before install.
- On stock/base installs, the app can identify and restore stock slot occupants.
- On Starter Character Pack installs, the app restores the previous slot 4 and slot 5 characters instead of assuming stock names.
- On unknown/custom installs, the app still restores the exact previous slot contents from the app-created backup.
